## 核心功能实现

### 1. 5 状态状态机

```python
State.RECEIVE   → 接收请求，验证输入
    ↓
State.EXTRACT   → LLM 并行抽取（5 维度）
    ↓
State.VALIDATE  → 完整性校验 + 质量评分
    ↓
State.GENERATE  → 模板填充（标准版/精简版）
    ↓
State.ARCHIVE   → 飞书文档创建/本地存储
```

### 2. 降级策略（5 级）

```
Level 0: 正常流程（qwen3.5-plus + 标准版）
    ↓ 超时/失败
Level 1: 重试 3 次（指数退避）
    ↓ 仍失败
Level 2: 降级 qwen2
    ↓ 抽取为空
Level 3: 追问用户→标记待确认
    ↓ 飞书失败
Level 4: 本地 Markdown 存储
    ↓ 成本超支
Level 5: 建议精简版→中止
```

### 3. 成本优化

| 策略 | 预期节省 |
|------|----------|
| 模板分级（精简版） | -50%（简单需求） |
| 缓存复用 | -15% |
| 批量处理 | -10% |
| 模型降级 | -75%（qwen3.5→qwen2） |

### 4. 安全防护

- ✅ Prompt Injection 过滤
- ✅ 输入长度限制（10000 字）
- ✅ 文件类型验证
- ✅ 敏感信息过滤
